# MeMusicPad

I made a HackPad that is meant to control my music! MeMusicPad is a 5 Key Marcopad with a rotary encoder. it runs off the KMK Firmware

## Features
- 5 Keys (4 for controlling music, and 1 for programmable use)
- Rotary Encoder for controlling volume
- LCD for whatever you want to do with

**Here are the things you will need to make this hackpad**

- 5x Cherry MX Switches
- 5x DSA Keycaps
- 2x heatset inserts
- 1x EC11 Rotary Encoder
- 1x XIAO RP2040
- 1x Case (2 printed parts)
